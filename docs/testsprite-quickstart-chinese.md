# TestSprite 快速入门指南（中文版）

> 面向华语开发者的 TestSprite 官方快速入门文档中文翻译

## 什么是TestSprite？

TestSprite是AI驱动的自动化测试平台，能自动生成、运行和维护测试用例。无需手动编写测试代码。

## 核心优势

- **零代码测试**：用自然语言描述场景，AI生成测试用例
- **自我修复**：UI变更时测试自动更新，无需手动维护
- **真实浏览器**：在Chrome/Firefox中运行，结果可靠

## 快速开始（10分钟上手）

### 第一步：注册

1. 访问 testsprite.com
2. GitHub或邮箱注册（推荐GitHub一键登录）
3. 免费套餐包含5个项目 + 每月500次运行

### 第二步：创建第一个测试

**方法A（推荐新手）——自动扫描：**
1. 点击「新建测试」
2. 输入应用URL
3. 等待AI扫描（约90秒）
4. 确认生成的测试用例后点击运行

**方法B——自然语言描述：**

在编辑器中描述测试步骤（支持中英文）：

```
测试：用户登录流程
步骤：
1. 打开登录页面
2. 输入邮箱 test@example.com
3. 输入密码
4. 点击登录按钮
5. 验证跳转到用户主页
```

### 第三步：查看报告

测试完成后自动生成报告，包含：
- 每步骤截图（高清）
- 完整执行视频
- Bug详情和复现步骤
- 时间线分析

### 第四步：CI/CD集成

**GitHub Actions示例：**

```yaml
name: TestSprite CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: testsprite/action@v1
        with:
          api_key: ${{ secrets.TESTSPRITE_KEY }}
          project_id: your-project-id
```

### 第五步：开启自动维护

设置 → 自动维护 → 开启

开启后，UI变更时AI自动修复测试用例，无需人工干预。

## 常见问题

**Q：支持哪些编程语言？**
A：无代码平台，不依赖特定语言。SDK支持JavaScript/Python/Java。

**Q：能测试需要登录的页面吗？**
A：支持。在「认证设置」配置登录凭据即可。

**Q：数据安全吗？**
A：HTTPS加密传输，测试凭据存储在加密密钥管理系统中。
